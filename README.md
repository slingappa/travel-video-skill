# travel-video skill

Claude Code slash command + Remotion template for generating polished 5-minute travel videos from a folder of trip photos and videos.

## Features

- Ken Burns effect on photos (portrait-aware — no cropping vertical shots)
- Animated day title cards with hollow text + zoom punch-through exit
- Trimmed video clips (auto-selected by file size, or pick via GUI)
- Optional animated map scenes (MapLibre + CARTO free tiles — no API key)
  - Flight arcs (geodesic), car/boat routes
  - City landmark SVG icons with bounce-in animation
  - Camera tracks line tip zoomed in per transport mode
- Interactive photo+video picker GUI (tkinter)
- Auto portrait detection → contain + dark letterbox
- Music with fade-in + fade-out

## Install

```bash
git clone <this-repo> ~/travel-video-skill
cd ~/travel-video-skill
bash install.sh
```

`install.sh` copies the slash command to `~/.claude/commands/travel-video.md`.

## Usage

In any Claude Code session:
```
/travel-video
```

Claude will ask you for trip details interactively, scan your media for EXIF dates, and generate the full Remotion project.

## Prerequisites

- Node.js 18+, npm
- `ffmpeg` — `brew install ffmpeg`
- Python 3: `pip install exifread pillow`

## Template structure

```
template/
  src/
    components/
      KenBurnsPhoto.tsx    ← Ken Burns with portrait support
      VideoClip.tsx        ← trimmed video with fade
      DayTitleCard.tsx     ← hollow text + zoom punch-through
    scenes/
      IntroScene.tsx       ← customize: title, dates, location
      OutroScene.tsx       ← customize: closing line, stats
      DayScene.tsx         ← generic per-day assembler
      map/
        JourneyMap.tsx     ← core MapLibre animated map
        CityOverlay.tsx    ← SVG city landmark icons
        mapData.ts         ← customize: cities, journey legs
        JourneyArrival.tsx ← customize: city positions
        JourneyDeparture.tsx
    TripVideo.tsx          ← customize: MAP_BEFORE dates
    Root.tsx               ← Remotion root composition
    data/
      mediaManifest.ts     ← auto-generated by generateManifest.py
  scripts/
    generateManifest.py    ← scans EXIF/ffprobe → manifest
    patch-orientations.py  ← detects portrait photos → patches manifest
    picker.py              ← GUI: select photos + videos per day
command/
  travel-video.md          ← the slash command (installed by install.sh)
install.sh                 ← installs command to ~/.claude/commands/
```

## Render

```bash
# Preview
npx remotion studio

# Render (software WebGL — no GPU needed)
node_modules/.bin/remotion render TripVideo --gl=swangle --output out/trip.mp4
```
